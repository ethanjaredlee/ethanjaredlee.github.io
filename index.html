Projects

  • Infinite Jukebox [code] My friend and I found this really cool thing made my folks at MIT called Infinite Jukebox that allowed a user to infinitely loop a song by identifying similar beats and jumping between them before the song reached the end. This allowed for a seamless transition and an almost unnoticable jump. The code attached is the raw code we used in development, but for cleanliness sake, I’m working to clean it up and make it easier to read and hopefully learn from for other people.
  • Squatters [code] This was one of the first scripts I worked on that I’m pretty sure nobody else in the world has ever done. I’d call this the turning point of where I decided that I wanted to be a CS major. The motivation of it comes from the fact that Northwestern Library rooms are reservable for up to 3 hours, but they always fill up during finals week (when the library is open 24 / 7). So fairly quickly my friends and I realized if you had 8 people you could indefinitely reserve a library room. This script automates that process. It’s fairly outdated and hasn’t been in use since that one quarter, but it was a lot of fun to do and a big reason why I’m CS now.
  • Tsuro Implementation of the Tsuro board game for Northwestern Software Construction. It’s a pretty interesting project where we focused a lot on how to abstract and break down the problem while adhereing to TDD. The implementation allows for a player, connected players over a network, and AI players. Currently the code itself is private in a school repo, but I’m trying to see if I can make it public.
